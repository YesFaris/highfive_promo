Au cours de de cette formation nous allons avoir 2 matières :

L'intégration

Le HTML :

Le HTML sert à creér le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site: les couleurs, les formes, le fond, etc.

Le développement

Le Javavscript

Javavscript est un langage de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramètres.

Le PHP:

Le PHP est un langage de programmation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc.)

MySQL:

MySQL est un système de gestion de base de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple: formualire de création de compte) pour ensuite être ré-utilisé au moment appoprié (ex: connexion à un site internet avec identifiant et mot de passe).
